# ChocoFerris – Chocolate Store Website Plan

## 1. Project Overview
**Website Name:** ChocoFerris  
**Purpose:** Online chocolate shopping platform with gifting, customization, and festive offers.  
**Target Audience:** Kids, teenagers, families, gift buyers, chocolate lovers.

---

## 2. Core Features

### 2.1 User Features
- User Registration & Login
- Guest Checkout
- User Profile & Order History
- Wishlist / Favorites
- Reviews & Ratings
- Newsletter Subscription

### 2.2 Product Features
- Product Categories
- Product Images (High Quality)
- Price, Description, Ingredients
- Stock Availability
- Related Products
- “Add to Cart” and “Buy Now”

### 2.3 Smart Filters
- Price Range
- Chocolate Type (Dark / Milk / White)
- Flavors (Caramel, Nuts, Fruit, Mint, etc.)
- Dietary Options (Sugar-Free, Vegan)
- Occasion (Birthday, Anniversary, Festival)
- Popularity / Ratings
- New Arrivals
